7- Limiter les résultats avec le mot-clef LIMIT

  • Afficher les employés en les classant par salaire (du + grand au plus petit) et en les affichant par 3 SELECT nom, prenom, service, salaire FROM employes ORDER BY salaire DESC LIMIT 3;
  • Le premier chiffre précise l'enregistrement de départ (0 définit le 1 eer enregistrement en informatique), le 2nd chiffre nous indique le nombre de résultats que l'on souhate obtenir
  • Autre ex. : SELECT nom, prenom, service, salaire FROM employes ORDER BY salaire DESC LIMIT 5,3;
  • Le mot-clef ou la clause LIMIT est utile pour faire une pagination web

Compter avec COUNT

  • Compter le nombre d'employées : SELECT COUNT(*) FROM employes WHERE sexe='f'; ou SELECT COUNT(*) FROM employes WHERE sexe='m';

  • Trouver le saliare minimum : SELECT MIN(salaire) FROM employes; pour trouver le salaire le plus bas et la personne qui gagne ce salair il faut écrire SELECT prenom, salaire FROM employes WHERE salaire = (SELECT MIN(salaire) FROM employes); : Le système va d'abord exécuter la requête entre parenthése pour connaître le resultat et l'utiliser comme source pour l'autre requête
  • Trouver le saliare maximum : SELECT MAX(salaire) FROM employes; Une requêtes embrquer
  • Par contre cette requête ne fonctionnera pasSELECT prenom, MIN(salaire) FROM employes;
  • Trouver qui gagne le plus : SELECT MAX(salaire) FROM employes;
  • Trouver qui gagne le minus : SELECT nom, prenom, salaire FROM employes WHERE salaire = 1390;/li>

  • IN permet d'annoncer plusieurs valeurs'

Conditions multiples, différentsb champs et valeurs

  • Les employés qui travaillent qu service commercial et qui gagne un salaire inférieur ou égal à 2000 EurosSELECT nom, prenom, salaire, service FROM employes WHERE service = 'commercial' AND salaire <= 2000;

Ordre de priorité sur les condition AND et OR

  • SELECT nom, prenom, service, salaire FROM employes WHERE service = 'production' AND salaire = 1900 OR salaire = 2300;
  • Lorsqu'il y a AND et OR au sein de la même requête, le système lit la condition de la droite vers la gauche et il verra d'abord salaire = 1900 or salaire = 2300 et ensuite service = 'production'
  • SELECT nom, prenom, salaire FROM employes WHERE salaire IN (1900, 2300) AND service='production';
  • Pour garantir l'ordre naturel de lecture (de la gauche vers la droite) on utilisera les parenthése : SELECT prenom, nom, service, salaire FROM employes WHERE service='production' AND (salaire= 1900 OR salaire=2300);

Les regroupements avec GROUP BY

  • Si on veut connaître le nombre d'emplotés par service : SELECT service, COUNT(*) AS nombre_employe FROM employes GROUP BY service;

Utilisation du mot-clef HAVING, ayant

  • On impose une condition COUNT et HAVINGSELECT service, COUNT(*) AS nombre FROM employes GROUP BY service HAVING COUNT(*) > 2;

Les requêtes de modification

La requête UPDATE

  • Augmenter le salaire d'un employé : UPDATE employes SET saliare = 1871 WHERE id_employes = 990;
  • Il est recommandé de modifier une enregistrement en ciblant son id, l'id étant unique
  • ....
  • Pour modifier plusieurs colonnes respectons l'ordre des colonnes : UPDATE employes SET service = 'commercial', 'saliare' = 1999 WHERE id_employes = 990;

La requête DELETE

  • Supprimer un enregistrement ; DELETE FROM employes WHERE id_employes = 388;
  • Supprimer plusieurs enregistrements : DELETE FROM employes WHERE service = 'informatique';
